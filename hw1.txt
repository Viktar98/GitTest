1.  Как скопировать на свой компьютер репозиторий из github.com,
    что должно быть сделано предварительно?

 git clone <ссылка на репозиторий на GitHub>
 Для последующей работы (push), компьютер и удалённый сервер (github)
 должны быть связаны ssh-ключом.

3.  Как получить на локальный компьютер изменения с github.com

 git pull - для единовременного "подтягивания" изменений и модификации файлов
 в локальном репозитории.

2. Как создать git репозиторий из локального каталога на своем компьютере.
   Как его отправить на github.com?

 git init - создаёт внутри скрытый репозиторий для версионного контроля, тем
 самым превращает репозиторий в git-репозиторий.

4. Какую информацию нам дают команда :

 git status - показывает информацию о репозитории и состоянии файлов в ней.
 git log - показывает список коммитов с информацией о них (время, автор, хеш и комментарий)
 git remote -v - показывает привязку локального репозитория и удаленного, в 
 случаи наличия такой привязки.
